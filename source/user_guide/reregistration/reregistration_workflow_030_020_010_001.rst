=================================================================
**"Contact Information" is missing.** [_patient_aux_verification]
=================================================================

    Essa situação ocorre quando as Informações de Contato do Paciente não estão disponíveis e o campo :bi:`Contact Information is unavailable` não foi marcado para indicar esse fato.

    :bi:`Verification Outcome`:

        * *Model Name*: **clv.patient_aux**
        * *Action*: **_patient_aux_verification**
        * *State*: **Error (L0)**
        * *Outcome Information*: '**"Contact Information" is missing.**'

    Essa situação é corrigida, indicando o campo :bi:`Contact Information is unavailable` como **marcado**.
