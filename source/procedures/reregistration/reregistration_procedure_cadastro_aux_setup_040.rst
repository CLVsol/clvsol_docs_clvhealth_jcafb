==============================================================
Associar todos os Endereços a um Endereço (Aux) (Procedimento)
==============================================================

    * *Workflow*: ":doc:`/reference_guide/reregistration/reregistration_cadastro_aux_setup`".

    #. Executar a Ação :bi:`Address Associate to Address (Aux)` para todos os Endereços:

        #. Acessar a *view* :bi:`Addresses`:

            * Menu de acesso:

                * :bi:`Community` » :bi:`Community` » :bi:`Addresses`

        #. Ativar o filtro **Agrupar por** » :bi:`Register State`

        #. Selecionar todos os Endereços com :bi:`Register State` != ":bi:`Canceled`"

        #. Executar a Ação :bi:`Address Associate to Address (Aux)`:

            * Parâmetros utilizados:

                * *Create new Address (Aux)*: **marcado**

            #. Utilize o botão :bi:`Associate to Address (Aux)` para executar a Ação.
