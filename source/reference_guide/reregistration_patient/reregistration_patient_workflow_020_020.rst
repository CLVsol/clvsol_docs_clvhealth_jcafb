==============================================================
Criação do **Cadastro Auxiliar** (**Paciente não cadastrado**)
==============================================================

O processo de criação do **Cadastro Auxiliar** para um Paciente **não cadastrado** é utilizado quando o Paciente tiver sido identificada como **não cadastrado**.

Portanto, quando não existir ainda um **Cadastro Auxiliar** para o Paciente, o mesmo deverá ser manualmente criado para conter as informações disponíveis para o Paciente. Essas informações serão manualmente inseridas nos registros pertinentes.

Se porventura já existir um **Cadastro Auxiliar** associado ao Paciente, o mesmo será selecionado para dar proseguimento ao processo de recadastramento do mesmo.

O **Cadastro Auxiliar** criado será composto pela **Entidade (Aux)**:

    * :bi:`Patient (Aux)`:

        * O registro será preenchido com as informações coletadas para o Paciente.
