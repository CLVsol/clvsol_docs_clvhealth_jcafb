=============================================================
**"Family" should not be set.** [_person_verification_family]
=============================================================

    O registro *Person* está associado a um registro *Family* e o campo *Family is Unavailable* está marcado.

    :bi:`Verification Outcome`:

        * *Model Name*: **clv.person**
        * *Action*: **_person_verification_family**
        * *State*: **Error (L0)**
        * *Outcome Information*: '**"Family" should not be set.**'

    Esta situação é corrigida, utilizando o procedimento de correção: :doc:`/procedures/reregistration/reregistration_procedure_040_030_015`.
